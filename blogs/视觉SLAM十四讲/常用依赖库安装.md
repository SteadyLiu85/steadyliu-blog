## 一、 基础

### 1. CMake 
```bash
# 安装基础编译工具
sudo apt-get install build-essential

# 下载并解压 CMake 3.16源码
wget [http://www.cmake.org/files/v3.16/cmake-3.16.0.tar.gz](http://www.cmake.org/files/v3.16/cmake-3.16.0.tar.gz)
tar xf cmake-3.16.0.tar.gz
cd cmake-3.16.0

# 编译与安装
./configure
make -j4
sudo make install

# 解决路径优先级问题，写入环境变量
sudo gedit ~/.bashrc
# 在文件末尾追加：
# export PATH=/usr/local/bin:$PATH
# export LD_LIBRARY_PATH=/usr/local/lib:$LD_LIBRARY_PATH

# 刷新环境变量并验证
source ~/.bashrc
cmake --version
```

### 2. Eigen 

**纯头文件库 (Pure Header Library)**，编译过程极快，因为只涉及头文件的拷贝。

```
wget [https://gitlab.com/libeigen/eigen/-/archive/3.3.9/eigen-3.3.9.zip](https://gitlab.com/libeigen/eigen/-/archive/3.3.9/eigen-3.3.9.zip)
unzip eigen-3.3.9.zip
cd eigen-3.3.9
mkdir build && cd build
cmake ..
make
sudo make install
```

### 3. Sophus 

相机的三维空间运动属于 $SE(3)$ 和 $SO(3)$ 群。由于旋转矩阵不满足加法封闭性，无法直接求导。Sophus 基于 Eigen 提供了李代数运算。

```
git clone [http://github.com/strasdat/Sophus.git](http://github.com/strasdat/Sophus.git)
cd Sophus
mkdir build && cd build
cmake ..
make
sudo make install
```

> 新版 Sophus 已经全面切换为模板类。在 C++ 源码中调用时：
>
> - 包含路径需更改：`#include <sophus/se3.hpp>` 和 `#include <sophus/so3.hpp>`。
>- 数据类型需去掉后缀：过时的 `SE3d` 与 `SO3d` 已废弃，必须使用模板实例化类 `SE3` 与 `SO3`（通常是指代 `SE3<double>`）。

## 五、 注意事项

在执行 `make` 编译 OpenCV 和 PCL 等大型库时，直接使用 `make -j`易导致内存泄漏从而引发系统死机。

通常每个编译线程需要 1.5GB - 2GB 的 RAM。如果你的虚拟机或物理机只有 8GB 内存，切记严格使用 `make -j4`；如果内存 $\le$ 4GB，请退回保守的 `make -j2` 甚至 `make`。在 CMakeLists.txt 的工程体系中，稳妥编译是最高优先级。
